COMPUTER

A computer system is divided into two categories: Hardware and Software. Hardware refers to the physical and visible components of the system such as a monitor, CPU, keyboard and mouse. Software, on the other hand, refers to a set of instructions which enable the hardware to perform a specific set of tasks.

HARDWARE

Software Hardware
Software is a program that enables users to interact with the computer, its hardware. Hardware is a physical component of computers that executes the instruction.
It is developed by software programmers or software development companies. It is manufactured in factories.
Operating Systems, Application Software, and Programming Software are the main categories of software. Storage Devices, Input Devices, Output Devices, and Internal components are the primary categories of hardware.
The software can be seen but cannot be touched as it is virtual, not physical. Hardware can be seen and touch as it is a physical, electronic device.
Computer viruses can affect software. Computer viruses cannot affect hardware.
Examples of software are Google Chrome, MySQL, MS Word, Excel, PowerPoint, Notepad, Photoshop and more. Examples of hardware are RAM, ROM, Printer, Monitor, Mouse, Hard disk and more.
